#### Macを使う場合の注意点
macOS Catalina 10.15 (以降も?) でUSB接続のカメラを使う場合公式のVideoライブラリーではおそらくエラーとなる。
先人の某ブログ
https://note.com/mement_mori1202/n/n4e888e7ef8b9
の手順でターミナル経由で実行すル必要があるため注意すること。
※ 10.14以前は(多分)問題ない
